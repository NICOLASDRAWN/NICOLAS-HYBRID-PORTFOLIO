│  🛡️ SEGURIDAD Y BACKUPS                                                  │
└────────────────────────────────────────────────────────────────────────────┘

  BACKUPS AUTOMÁTICOS:
  • Se crean al iniciar el sistema
  • Ubicación: backups/backup_YYYY-MM-DD_HHMM/
  • Incluyen: Usuarios, Proveedores, Productos, Órdenes, Configuración

  BACKUPS MANUALES:
  • Opción 3 del menú interactivo
  • O ejecutar BACKUP_RAPIDO.bat
  • Ubicación: backups/manual_YYYY-MM-DD_HHMM/

  CONTENIDO DE CADA BACKUP:
  ✓ Base de datos completa (backend/data/)
  ✓ Archivos de configuración (*.json)
  ✓ Archivo INFO.txt con detalles

  RECOMENDACIONES:
  • Hacer backup diario (automático al iniciar)
  • Hacer backup manual antes de cambios importantes
  • Mantener al menos 7 días de backups
  • Copiar backups importantes a otra ubicación
